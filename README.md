# Alpine Base Docker Image
Yet another alpine image contains 

- Bash
- AWS Cli
- Azure Cli 
- Telnet / SSH
- DNS Tools
- Nmap


Run Docker Container
-----------------

    docker run -it ssarioglu/alpine
